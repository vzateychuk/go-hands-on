# Mongo Env
Using environment variables to set up or skip tests

## Mongo DB

Start MongoDB with Mongo-Express
```shell
docker compose up
```

Open MongoExpress (connecto to mongoDB)
```shell
http://localhost:9081/
```

## Environment

```shell
DB_HOST=127.0.0.1;DB_PORT=27017;DB_USER=...;DB_PASSWORD=...
```
